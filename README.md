# Fortune Advisory — Financial Screener

A Streamlit dashboard for analyzing Vietnamese listed companies' financial statements — powered by vnstock data. Visualize key financial ratios, benchmark any company against its industry peers, and screen entire industries.

---

## What it does

| View | Description |
|------|-------------|
| **Company Dashboard** | Time-series charts of 17 financial ratios across 4 quarters, overlaid with industry average |
| **Company vs Industry** | Benchmark table: company vs Avg Industry / Top 1 / Top 2 / Top 3 / Avg Top 3 |
| **Industry Screener** | Filter by industry — sortable company table, bar chart, multi-company trend |

---

## Prerequisites

- **Python 3.8+** — verify with `py --version` in your terminal
- **One Excel file** generated by the companion notebook (`vnstock_pull_bctc.ipynb`):
  - `ALL_BCTC_KBS_quarter_4Q_[timestamp].xlsx` — IS + BS + CF for all listed companies, with industry column included

## Setup (one time)

### Step 1 — Prepare the SSI industry file (one time)

Download the sector classification from SSI and save as **`ssi_industry.xlsx`** in the **same folder as the notebook** with this exact format:

| Sheet name | Column A | Column B |
|------------|----------|----------|
| `TỔNG QUAN` | `MÃ` | `NGÀNH` |

Example rows:
```
MÃ      NGÀNH
ACB     Ngân hàng
VNM     Thực phẩm & Đồ uống
HPG     Thép
```

This file only needs to be updated when SSI revises its sector classification (typically once or twice a year).

### Step 2 — Generate the data file (run once, ~2 hours)

Open `vnstock_pull_bctc.ipynb` in Jupyter and run all cells top to bottom.

`OUTPUT_DIR` defaults to `'.'` (same folder as the notebook), so the generated file lands alongside all other project files. Change it in **Cell 3** only if you want output elsewhere:

```python
OUTPUT_DIR = '.'   # default — keeps everything in one folder
```

The notebook will:
1. Read `ssi_industry.xlsx` from the same folder to build the industry map
2. Pull IS + BS + CF for all ~1,500 listed companies across 4 quarters from KBS
3. Embed the `industry` column into the output
4. Export `ALL_BCTC_KBS_quarter_4Q_[timestamp].xlsx` to `OUTPUT_DIR`

Cell 8 prints industry coverage stats before saving — if coverage is below ~80%, update `ssi_industry.xlsx` and re-run.

> **If `ssi_industry.xlsx` is missing:** the notebook still runs and exports, but `industry` will be empty and the Industry Screener / peer benchmarking views in Streamlit will not work.

### Step 3 — Install dashboard dependencies

```bat
py -m pip install -r requirements.txt
```

Or double-click **`setup.bat`** on Windows.

### Step 4 — Run the dashboard

`config.py` defaults to `RAW_EXCEL_DIR = "."`, so it automatically finds the generated Excel file if everything is in the same folder. No edits needed unless your data file is elsewhere.

```bat
py -m streamlit run streamlit_app.py
```

Or double-click **`run.bat`**.

The app opens automatically at `http://localhost:8501`.

## Fixing missing columns (Column Check)

The app sidebar has a **Column Check** panel. If it shows `MISSING` for any column, your vnstock output uses different column names.

**Find your actual column names:**

```python
import pandas as pd, glob
f = sorted(glob.glob(r"YOUR_RAW_EXCEL_DIR\ALL_BCTC_*.xlsx"))[-1]
print(pd.read_excel(f, sheet_name="TONG_HOP", nrows=0).columns.tolist())
```

Then update the matching variable in `config.py` (Section 3) and restart the app.

## Architecture

```
vnstock API (KBS source)
    |
    v
vnstock_pull_bctc.ipynb
    |  pulls IS + BS + CF for all listed companies
    |  4 quarters, ~1,500 tickers
    |  industry column added manually from SSI classification export
    |
    v
ALL_BCTC_KBS_quarter_4Q_[ts].xlsx   ← single file, no other dependency
    |  TONG_HOP sheet: ticker / exchange / year / quarter / industry + ~750 financial cols
    |
    v
streamlit_app.py
    |
    +------------+------------+
    |            |            |
Company      Company      Industry
Dashboard  vs Industry    Screener
    |            |            |
time series  benchmark    sortable
4 quarters   table 7-col  table +
+ industry   + bar chart  bar chart
avg overlay               + trend
```

**Ratio computation:** 17 ratios computed on-the-fly in pandas. Quarterly IS/CF items are annualised x4. Division-by-zero and NaN/Inf values are handled gracefully (display as `-`).

**Industry benchmarks:** For each ratio, the app groups all companies in the same industry and same quarter, then computes: Avg, Top 1 / Top 2 / Top 3 (ranked by higher-is-better direction), Avg Top 3.

## Known limitations

- **Column names**: KBS vnstock output naming may change across versions. The Column Check panel and `config.py` section 3 handle this.
- **EBITDA**: approximated as EBIT + D&A. If `depreciation_amortization` column is missing, EBITDA = EBIT.
- **Quarterly annualisation**: IS and CF ratios use period value x4, not rolling TTM. This overstates/understates for seasonal businesses.
- **Industry classification**: sourced manually from SSI's sector mapping export — KBS does not provide complete classification. Newly listed companies or reclassified tickers will be missing until the next SSI export is applied.
- **File size**: the raw Excel is ~50MB. First load takes 10-20 seconds; subsequent loads use Streamlit cache.
